
# Mentoria Gupy backend

O projeto esta sendo criado, utilizando o conhecimento passado através da mentoria do Daniel Tamai.
A proposta desse projeto é aumentar meus conhecimento com backend, utilizando docker.

## Intalação e execução
Baixe o projeto e rode os seguintes comandos:
```
- git clone git@github.com:IzauBarbosa/mentoria-gupy-backend.git
- cd mentoria-gupy-backend
- cd backend && docker build -t izaubarbosa/mentoria-gupy-backend .
- docker-compose up
```
Após rodar os comandos acima, você pode visualizar a aplicação rodando em `http://localhost:8000/`
